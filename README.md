# ForterSDK ReactNative wrapper

We provide this wrapper for you to be able to maintain a single 
point of initialization for your app, instead of doing native 
initialization of our SDK in IOS and Android.

This plugin is built for:
- Forter's iOS SDK
- Forter's Android SDK

## Installation
Add the Forter dependency to your `package.json`:

```json
{
 "dependencies": {
       "react": "16.8.6",
       "react-native": "0.60.4",
       "react-native-forter": "git+https://forter-mobile-git:zvGKcVtDhkfj4asNekSn@bitbucket.org/forter-mobile/forter-react-plugin.git#v0.14"
  },
}
````

### iOS specific implementation

1. Add the `ForterSDK` pod and the React Native pod to your `Podfile` and run `pod install`.

Example, `ios/Podfile`

``` podspec
pod 'ForterSDK', :git => 'https://forter-mobile-git:zvGKcVtDhkfj4asNekSn@bitbucket.org/forter-mobile/forter-ios-sdk.git'
pod 'react-native-forter', :path => '../node_modules/react-native-forter'
```

You must also have the React dependencies defined in the Podfile as described [here](https://facebook.github.io/react-native/docs/next/troubleshooting.html#missing-libraries-for-react).

2. Run `pod install` (inside `ios` directory).


### Android 

##### **android/app/build.gradle**


Run `react-native link react-native-forter` from of the project root or add manually:

Add the project to your dependencies
```gradle
dependencies {
...
compile project(':react-native-forter')
}
```

##### **android/settings.gradle**

Add the project

```gradle
include ':react-native-forter'
project(':react-native-forter').projectDir = new File(rootProject.projectDir, '../node_modules/react-native-forter/android')
```

Add this to `android/app/build.gradle` to our private Maven repository:
``` gradle
repositories {
  maven {
    url "https://mobile-sdks.forter.com/android"
    credentials {
      username "forter-android-sdk"
      password "HvYumAfjVQYQFyoGsmNAefGdR84Esqig"
    }
  }
}
```

##### **MainApplication.java**

If you are using ReactNative >= 0.60, the package list is autogenerated.

If you are running ReactNative <= 0.59, add the following code to your app:

1. `import com.forter.mobile.reactnative.RNForterPackage;`

2.  In the `getPackages()` method register the module:
`new RNForterPackage(MainApplication.this)`

So `getPackages()` should look like:

```java
    @Override
    protected List<ReactPackage> getPackages() {
      return Arrays.<ReactPackage>asList(
            new MainReactPackage(),
            //...
            new RNForterPackage(MainApplication.this)
            //...
      );
    }
```
## JavaScript setup

Add the following code to your apps `index.js`:

``` javascript
import {logger} from 'react-native-logger';
import {forterSDK, ForterActionType, ForterNavigationType} from 'react-native-forter';

AppRegistry.registerComponent(appName, () => App);

// Modify this with your merchant ID
var myForterID = "1234556789" 

forterSDK.getDeviceUniqueID( (deviceID) => {
    forterSDK.init(myForterID, deviceID, (successResult) => {
        logger.log("OK: " + successResult);
    }, (errorResult) => {
        logger.log("FAIL: " + errorResult);
    });
})

// Exaples for custom tracking
forterSDK.trackNavigation('mainpage', ForterNavigationType.PRODUCT);
forterSDK.trackAction(ForterActionType.ACCOUNT_LOGIN)
```
